* Python 有五个基础类型

  * Numbers （数字）
    *  int （整型）
    * long (长整型) 需要在后面加上`L`进行标记
    * float （单精度浮点）
    * complex （复数）
  * String （字符串）
  * List （列表）
    * List类似c语言中的数组
  * Tuple （元组）
  * Dictionary （字典）
